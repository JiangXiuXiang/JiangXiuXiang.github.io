## 3. AR开发

3.1 三行代码开发AR（入门示例）

```swift
// 程序窗口加载时
sceneView.scene = SCNScene(named: "ship.scn", inDirectory: "models.scnassets/ship")!
// 程序窗口显示时
sceneView.session.run(ARWorldTrackingConfiguration())
// 程序窗口隐藏时
sceneView.session.pause() 
```
